# LiveBG
A Wayland baised project that allows for the simple creation of a custom, interactive background for you Linux system.
## Features
* Window generation
* Rendering and ticking loop
* Keyboard input handling
* Multiple draw functions
* Fullscreen and windowed boarderless toggle: ``F11``
* Exit keybind: ``Esc``
* Various utility functions
* Documentation!!!!!!
